# خطوات تشغيل البوت :
## 1 :
> قم باضافة المعلومات المناسبة في ملف ``.env`` :
```js
mongo = "رابط المونجو"
SystemToken = "توكن بوت السيستم"
giveawayToken = "توكن بوت الجيف اوي"
tachhirToken = "توكن بوت التشهير"
```
## 2 :
> املأ جميع المعلومات المطلوبة في ملف ``config.js``

## 3 :
> توجه لمجلد database

> املأ جميع المعلومات المطلوبة في ملف ``settings.json``

> املأ جميع المعلومات المطلوبة في ملف ``prices.json``
